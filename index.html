Добавить данные о поставщике для анализа в экспертной системе выбора победителя закупочной процедуры

Соответствие заявки и приложений ос-новным техническим требованиям доку-ментации о закупке.
Наиболее выгодными признаются пред-ложения, содержащие наименьшую цену договора (лота). Количество баллов, присуждаемых заяв-ке по критерию, определяется по форму-ле: Rai= {{Amax-Ai} / {Amax}} ×100, где: Rai– кол-во баллов, присуждаемых i-й заявке по указанному критерию; Amax - начальная (максимальная) цена до-говора; Ai- цена договора, предложенная i-м участником.
Наиболее выгодными признаются пред-ложения, содержащие наилучшее сочета-ние опыта работы на рынке по теме про-ведения процедуры закупки при разме-щении заказа, специфики работы органи-зации, уровня деловой репутации и про-фессиональной квалификации.
Качество поставляемого товара (выпол-нения работ, оказания услуг) определяет-ся исходя из предложений участников: - по форме подачи информации и деталь-ности проработки в соответствии с тре-бованиями технической части заявки; - по полноте и форме представления письменной информации по поставляе-мому товару (выполнению работ, оказа-нию услуг).
Наиболее выгодными признаются пред-ложения в заявке с наименьшим суммар-ным сроком исполнения договора в еди-ницах измерения (количество лет, квар-талов, месяцев, недель, дней, часов). Первое рейтинговое место по данному критерию присваивается Участнику, предложившему наименьший срок ис-полнения обязательств по договору, да-лее - по мере ухудшения выгодности предложений.
Объем представления гарантии качества товара (работ, услуг) (предмет и исчер-пывающий перечень условий исполнения гарантийного обязательства) содержится в документации к заявке. Наиболее вы-годными признаются предложения, обес-печивающие наиболее продолжительный срок гарантии качества товара (работ, услуг) с момента исполнения договора.
Наиболее выгодными признаются пред-ложения, содержащие наилучшее сочета-ние условий оплаты по исполнению до-говора: аванс, окончательный расчет, промежуточные и другие платежи в со-ответствии с требованиями, указанными в документации.
